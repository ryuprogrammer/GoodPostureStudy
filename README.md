## 3. アプリの動作
アプリの動作が分かる動画や、スクリーンショットを作成して掲載をしてください。
具体的にどんな操作をしているのか、直感的に分かるように操作をして示してください。

### 動画撮影の方法
Gif動画、mp4どちらかで撮影してください。
mp4であれば、静的なキャプチャの用に見えるので、クリックして再生することを明記して置く必要があります。
Gif動画であれば、再生ボタンをクリックしなくてもループして再生させることが出来ます。

- [mp4の動画取得方法](https://otona-life.com/2022/12/22/149228/)
- [GIPHY Capture](https://one-time-life.net/service/giphy-capture-mac-gif/)：Gif動画を作成するのに便利なアプリです。
